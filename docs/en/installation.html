Installation

The INDICATE Data Dictionary application can be installed locally as an R package or deployed using Docker.

Option 1: Install as R Package

Prerequisites

  • R version 4.0.0 or higher

Installation Steps

  1. Install from GitHub

    Open R and run:

    # Install remotes if not already installed
install.packages("remotes")

# Install the INDICATE Data Dictionary package (dev version, recommended)
remotes::install_github("indicate-eu/data-dictionary", ref = "dev")

# Or install the stable version from main branch
remotes::install_github("indicate-eu/data-dictionary")
    Development Version

    During active development, we recommend installing the dev branch to access the latest features and bug fixes.

  2. Launch the application 
    indicate::run_app()

    The application will open in your default web browser.

Configuration Options

You can customize the application behavior with these parameters:

indicate::run_app(
  language = "en",
  app_folder = NULL,
  debug_mode = "error",
  port = 3838,
  host = "0.0.0.0"
)
Parameter Description Default
language Interface language: "en" (English) or "fr" (French) "en"
app_folder Path to folder where application files will be stored. If NULL, uses user's home directory. NULL
debug_mode Debug output level: "error", "event", or c("event", "error") "error"
port Port number for the Shiny server 3838
host Host address ("127.0.0.1" for local only, "0.0.0.0" for network access) "0.0.0.0"

Option 2: Docker Deployment

Coming Soon

A Docker image will be available on DockerHub shortly. This will allow you to deploy the application in isolated environments without installing R.

Building from Source

In the meantime, you can build the Docker image locally:

# Clone the repository
git clone https://github.com/indicate-eu/data-dictionary.git
cd data-dictionary

# Build the image
docker build -t indicate-data-dictionary .

# Run the container
docker run -p 3838:3838 indicate-data-dictionary

Open your browser and navigate to http://localhost:3838
