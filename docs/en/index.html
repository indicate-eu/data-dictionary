Introduction

The INDICATE Data Dictionary is an interactive web application designed to explore and align clinical data with the INDICATE Minimal Data Dictionary - a consensus-based collection of 11,924 standardized clinical concepts for harmonizing intensive care unit (ICU) data across Europe.

What is INDICATE?

INDICATE (INtensive care Data Integration CollAborative TErminology) is a European collaborative project aimed at creating a standardized vocabulary for ICU data, enabling better interoperability and research across healthcare systems.

Key Features

Dictionary Explorer

Browse 11,924 standardized clinical concepts organized by category and subcategory.

OMOP Mappings

View mappings to standard terminologies: SNOMED CT, LOINC, RxNorm, and ICD-10.

Concept Mapping

Align your local source concepts with the INDICATE dictionary general concepts.

Projects

Organize concepts into projects for specific research use cases.

Export Formats

Export to Usagi, STCM, and ATLAS JSON formats for OHDSI workflows.

Expert Comments

Access expert guidance to help with concept alignment decisions.

Who is this for?

This application is designed for:

  • Data Scientists - Working on ICU data harmonization projects
  • Clinical Informaticists - Mapping local terminologies to standard vocabularies
  • Researchers - Exploring the INDICATE minimal data dictionary
  • OMOP CDM Users - Creating concept sets for observational research
  • Healthcare Data Engineers - Building ETL pipelines for clinical data

About the INDICATE Project

The INDICATE project brings together intensive care experts, clinical informaticists, and data scientists from across Europe to create a unified vocabulary for ICU data. The minimal data dictionary represents the consensus of this collaborative effort, providing a foundation for multi-center research and data sharing.

Each general concept in the dictionary is mapped to standard terminologies used in the OMOP Common Data Model, facilitating integration with the broader OHDSI (Observational Health Data Sciences and Informatics) network.
