Prise en main

Ce guide vous aidera à démarrer avec INDICATE Data Dictionary en quelques minutes. Suivez ces étapes pour explorer le dictionnaire et comprendre sa structure.

Étape 1 : Lancer l'application

Après l'installation, démarrez l'application :

indicate::run_app()

L'application s'ouvrira dans votre navigateur web par défaut. Consultez la page Installation pour les arguments disponibles (langue, port, etc.).

Étape 2 : Se connecter

Vous verrez l'écran de connexion. Utilisez les identifiants par défaut pour vous connecter :

  • Nom d'utilisateur : admin
  • Mot de passe : admin
Note de sécurité

Pour une utilisation en production, changez le mot de passe par défaut immédiatement après la première connexion via la section de gestion des utilisateurs.

Étape 3 : Configurer les vocabulaires OHDSI

À la première connexion, vous verrez un message d'erreur indiquant que les vocabulaires OHDSI ne sont pas chargés. Le dictionnaire nécessite ces vocabulaires pour fonctionner correctement.

Requis

Le téléchargement des vocabulaires OHDSI nécessite une inscription sur le site ATHENA et l'acceptation des accords de licence des vocabulaires.

  1. S'inscrire sur ATHENA

    Créez un compte sur athena.ohdsi.org

  2. Sélectionner les vocabulaires

    Sélectionnez les vocabulaires suivants pour le téléchargement : SNOMED-CT, LOINC, RxNorm, RxNorm Extension, UCUM, ATC, ICD-10.

  3. Télécharger et extraire

    Téléchargez le fichier ZIP et extrayez-le dans un dossier sur votre ordinateur. Notez le chemin vers ce dossier.

  4. Configurer dans l'application

    Cliquez sur l'icône de rouages en haut à droite de l'application, puis sélectionnez "General settings".

    Spécifiez le chemin vers le dossier contenant les fichiers CSV téléchargés depuis ATHENA (CONCEPT.csv, CONCEPT_RELATIONSHIP.csv, CONCEPT_ANCESTOR.csv, etc.).

Étape 4 : Explorer le dictionnaire

Une fois les vocabulaires chargés, vous pouvez accéder à la page Explorateur du dictionnaire. C'est l'interface principale pour parcourir le dictionnaire INDICATE.

Liste des concepts généraux

La vue initiale affiche un tableau avec tous les concepts généraux du dictionnaire.

  • Utilisez les badges de catégorie en haut pour filtrer par catégorie (Paramètres vitaux, Biologie, Traitements, etc.)
  • Utilisez le champ Recherche approx (en haut à droite) pour rechercher des noms de concepts avec correspondance approximative
  • Utilisez les filtres de colonnes pour les correspondances exactes uniquement
  • Double-cliquez sur une ligne ou cliquez sur le bouton View details pour voir les détails du concept

Détails d'un concept général

Lorsque vous sélectionnez un concept, vous accédez à la vue détaillée organisée en quatre panneaux :

  • En haut à gauche : Concepts associés - visualisez les concepts associés des terminologies OMOP (SNOMED, LOINC, RxNorm) pour ce concept
  • En bas à gauche : Commentaires experts - lisez les recommandations des experts cliniques pour aider à l'ETL et visualisez la distribution statistique attendue pour ce concept
  • En haut à droite : Détails du concept - consultez les détails du concept sélectionné, avec des liens vers ATHENA et le serveur de terminologie FHIR
  • En bas à droite : Concepts liés - visualisez les concepts liés, la hiérarchie (ancêtres/descendants) et les synonymes

Prochaines étapes

Maintenant que vous êtes familiarisé avec les bases, apprenez-en plus sur :

  • Dictionnaire de données - Explorez plus en détail la structure du dictionnaire, les concepts généraux et les commentaires experts
  • Projets - Créez des projets et assignez des concepts généraux à votre projet
  • Alignement de concepts - Alignez vos données sources avec les concepts du dictionnaire via le module d'alignement sémantique
