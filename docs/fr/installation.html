Installation

L'application INDICATE Data Dictionary peut être installée localement en tant que package R ou déployée via Docker.

Option 1 : Installation en tant que package R

Prérequis

  • R version 4.0.0 ou supérieure

Étapes d'installation

  1. Installer depuis GitHub

    Ouvrez R et exécutez :

    # Installer remotes si ce n'est pas déjà fait
install.packages("remotes")

# Installer le package INDICATE Data Dictionary (version dev, recommandée)
remotes::install_github("indicate-eu/data-dictionary", ref = "dev")

# Ou installer la version stable depuis la branche main
remotes::install_github("indicate-eu/data-dictionary")
    Version de développement

    Pendant la phase de développement actif, nous recommandons d'installer la branche dev pour accéder aux dernières fonctionnalités et corrections.

  2. Lancer l'application 
    indicate::run_app()

    L'application s'ouvrira dans votre navigateur web par défaut.

Options de configuration

Vous pouvez personnaliser le comportement de l'application avec ces paramètres :

indicate::run_app(
  language = "en",
  app_folder = NULL,
  debug_mode = "error",
  port = 3838,
  host = "0.0.0.0"
)
Paramètre Description Défaut
language Langue de l'interface : "en" (anglais) ou "fr" (français) "en"
app_folder Chemin vers le dossier où les fichiers de l'application seront stockés. Si NULL, utilise le répertoire personnel de l'utilisateur. NULL
debug_mode Niveau de débogage : "error", "event", ou c("event", "error") "error"
port Numéro de port pour le serveur Shiny 3838
host Adresse hôte ("127.0.0.1" pour local uniquement, "0.0.0.0" pour accès réseau) "0.0.0.0"

Option 2 : Déploiement Docker

Prochainement

Une image Docker sera disponible sur DockerHub prochainement. Cela vous permettra de déployer l'application dans des environnements isolés sans installer R.

Construction depuis les sources

En attendant, vous pouvez construire l'image Docker localement :

# Cloner le dépôt
git clone https://github.com/indicate-eu/data-dictionary.git
cd data-dictionary

# Construire l'image
docker build -t indicate-data-dictionary .

# Lancer le conteneur
docker run -p 3838:3838 indicate-data-dictionary

Ouvrez votre navigateur et accédez à http://localhost:3838

Vocabulaires OHDSI

L'application nécessite les vocabulaires OHDSI d'ATHENA pour fonctionner correctement. Ces vocabulaires permettent la recherche de concepts, la navigation dans les hiérarchies et la consultation des relations.

Requis

Le téléchargement des vocabulaires OHDSI nécessite une inscription sur le site ATHENA et l'acceptation des accords de licence des vocabulaires.

  1. S'inscrire sur ATHENA

    Créez un compte sur athena.ohdsi.org

  2. Sélectionner les vocabulaires

    Sélectionnez les vocabulaires suivants pour le téléchargement : SNOMED-CT, LOINC, RxNorm, RxNorm Extension, UCUM, ATC, ICD-10.

  3. Télécharger et extraire

    Téléchargez le fichier ZIP et extrayez-le dans un dossier sur votre ordinateur. Notez le chemin vers ce dossier.

  4. Configurer dans l'application

    Cliquez sur l'icône de rouages en haut à droite de l'application, puis sélectionnez "General settings".

    Spécifiez le chemin vers le dossier contenant les fichiers CSV téléchargés depuis ATHENA (CONCEPT.csv, CONCEPT_RELATIONSHIP.csv, CONCEPT_ANCESTOR.csv, etc.).
